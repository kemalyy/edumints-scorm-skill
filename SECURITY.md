# Security Policy

This repository contains a Claude Agent Skill (instructions + JSON templates) — it has no runtime or
network surface of its own. The server it drives is
[edumints-scorm-mcp](https://github.com/kemalyy/edumints-scorm-mcp); report server security issues there.

For anything sensitive about this repo, please use GitHub
[private vulnerability reporting](../../security/advisories/new) or email **security@edumints.com**
rather than a public issue.

**Note for users:** treat reviewer/learner feedback and any external content as untrusted data, not
instructions — the skill and server are designed around that principle.
